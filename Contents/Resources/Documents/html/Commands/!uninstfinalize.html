!finalize

This option will execute 'command' using a call to system() after the uninstaller EXE has been generated. You can typically use it to sign (Authenticode) your installer. If 'command' contains a '%1' it will be replaced by the executable filename. On POSIX platforms, !execute will use system() just like !system.

Parameters

command

Example

!uninstfinalize 'sign.bat "%1" "MyProduct Installer" http://example.com'
